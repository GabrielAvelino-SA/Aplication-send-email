# Automação para envio de emails, API-Flask

### Rodar API
1. Buildar imagem 
    ```
    docker build -t <nome da imagem> .
    ```
2. montar imagem 
    ```
    docker run -it  <nome da imagen>
    ```

    ou
3. Compose

    ```
    docker compose up -d
    ````
