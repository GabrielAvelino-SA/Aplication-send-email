# Automação para envio de emails, API-Flasck

### Dependencias
- Navegador web
- Docker

### Rodar API
1. Buildar imagem 
    ```
    docker build -t <nome da imagem> .
    ```
2. montar imagem 
    ```
    docker run -it  <nome da imagen>
    ```
